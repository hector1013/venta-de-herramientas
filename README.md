# Sitio web de venta  y alquileres de herramientas para construccion

Sitio web estático para mostrar herramientas y hacer pedidos por WhatsApp.

## Archivos

- `index.html` - Página principal.
- `styles.css` - Estilos del sitio.
- `script.js` - Lógica para abrir WhatsApp con pedido.

## Personalización

1. Cambia el número en `script.js` por tu número real de WhatsApp con código de país.
2. Reemplaza las imágenes y los productos con tus herramientas.
3. Abre `index.html` en un navegador para ver el sitio.
